1. 인증

1.1. 회원 가입

사용자는 회원 가입을 할 수 있습니다.

1.1.1. HTTP request

POST /get-p/v2/auth/signup HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 243
Host: api.principes.xyz

{
  "email" : "test@example.com",
  "password" : "qwer1234!",
  "verificationCode" : "1234",
  "serviceTerms" : [ {
    "tag" : "tag1",
    "agreed" : true
  }, {
    "tag" : "tag2",
    "agreed" : false
  } ],
  "memberType" : "ROLE_CLIENT"
}

1.1.2. Request fields

Path Type Required Enum Constraints Description

email

String

True

잘못된 이메일 형식입니다., 필수 입력 항목입니다.

이메일

password

String

True

필수 입력 항목입니다., 비밀번호는 8자 이상 20자 이하의 영문, 숫자, 특수문자를 포함해야 합니다.

비밀번호

memberType

enum

True

ROLE_PEOPLE ROLE_CLIENT ROLE_MANAGER ROLE_ADMIN

회원 유형. 일반 사용자는 피플인 ROLE_PEOPLE 또는 의뢰자인 ROLE_CLIENT만 가능

verificationCode

String

True

필수 입력 항목입니다.

이메일 인증 코드

serviceTerms[].tag

String

True

필수 입력 항목입니다.

서비스 약관 태그

serviceTerms[].agreed

Boolean

True

필수 입력 항목입니다.

서비스 약관 동의 여부

1.1.3. HTTP response

HTTP/1.1 201 Created
Content-Type: application/json;charset=UTF-8
Content-Length: 20

{
  "status" : 201
}

12.1. [회원] 파일 업로드

회원은 파일을 업로드할 수 있습니다. 업로드할 수 있는 파일 확장자는 pdf, zip, ppt, docx, hwp, jpg, png 입니다.

12.1.1. HTTPie request

$ http --multipart POST 'https://api.principes.xyz/get-p/v2/storage/files' \
    'Authorization:Bearer ${ACCESS_TOKEN}' \
    'file'='dummy'

12.1.2. Request headers

Name Description

Authorization

Bearer ${ACCESS_TOKEN}

12.1.3. Request parts

Part Description

file

업로드할 파일

12.1.4. HTTP response

HTTP/1.1 201 Created
Content-Type: application/json;charset=UTF-8
Content-Length: 105

{
  "status" : 201,
  "data" : {
    "fileUri" : "https://storage.principes.xyz/1/files/1/test.pdf"
  }
}
